Welcome to the Cyntacs Main Page!

What is it?

Cyntacs is a small, lightweight code editor written in python.

What can it do?

Cyntacs can:

  • open text-based files
  • highlight syntax based on a custom-defined "SyntaxDefs.txt" file, where you can specify what you want to highlight, and what color you want it to be
  • undo and redo
  • open files through the "open with..." command in windows
